# Phase 8, Plan 1: Signal Infrastructure & Death Watch

## Objective

Add a `Signal` delivery system to Bayou and implement death watch: any actor (or external code) can watch another for death and receive a `Terminated` signal when it dies — whether by crash or graceful stop.

This plan covers the infrastructure and watch/unwatch API. Linking and `trapExits` are in Plan 2.

## Context

**Key architecture facts:**
- `AbstractActorRunner.loop()` catches `Exception` around the entire while loop; anything thrown from message processing causes actor death with `terminalCause` set.
- The `finally` block currently: cancels timers, calls `cleanup()`, completes `stopFuture`, fires `crashListener` (only if `terminalCause != null`).
- `BayouSystem` has a `ConcurrentHashMap<String, Ref<?>> actors` (private). No runner map exists yet.
- `Envelope<M>` is a 2-field record: `(M payload, CompletableFuture<Object> replyFuture)`.
- `crashListener` fires ONLY on crash (unhandled exception). Graceful stop does NOT fire it.

**Design decisions for this plan:**

- **Signal delivery via mailbox:** Signals share the actor's mailbox (same `LinkedBlockingQueue`), preserving FIFO ordering. `Envelope` gains a `Signal signal` field and an `isSignal()` method.
- **Runners map in BayouSystem:** A second `ConcurrentHashMap<String, AbstractActorRunner<?>> runners` is added alongside `actors`. All `spawn*()` methods populate it. This avoids casting `Ref<?>` to get runner access, and stays package-private.
- **`signalListeners` in AbstractActorRunner:** `CopyOnWriteArrayList<Consumer<Signal>>` — thread-safe for concurrent watch/unwatch while rarely modified. Watchers add a consumer; death fires all consumers.
- **Death fires Terminated unconditionally:** Both crash path (`terminalCause != null`) and graceful stop path (normal loop exit) fire `Terminated` to all signal listeners in the `finally` block, AFTER `stopFuture.complete(null)`.
- **`handleSignal(Signal)` in AbstractActorRunner:** Non-abstract default (no-op) — only actor runners override it. `SupervisorRunner` inherits the no-op (supervisors don't have user-facing `onSignal`).
- **`WatchHandle` pattern:** Watch registration returns a `WatchHandle` (public interface with `cancel()`). `WatchHandleImpl` holds the target runner reference and the exact consumer to remove.

**Create `src/main/java/com/cajunsystems/bayou/WatchHandleImpl.java`** (package-private):
```java
package com.cajunsystems.bayou;

import java.util.function.Consumer;

final class WatchHandleImpl implements WatchHandle {
    private final AbstractActorRunner<?> target;
    private final Consumer<Signal> listener;

    WatchHandleImpl(AbstractActorRunner<?> target, Consumer<Signal> listener) {
        this.target = target;
        this.listener = listener;
    }

    @Override
    public void cancel() {
        target.signalListeners.remove(listener);
    }
}
```

**Add `watch`/`unwatch` to `BayouSystem`:**
```java
/**
 * Register a death watch on {@code target}. When the target actor stops (crash or graceful),
 * {@code watcher} receives a {@link Terminated} signal via its {@code onSignal} handler.
 *
 * @return a handle to cancel the watch
 */
public WatchHandle watch(Ref<?> target, Ref<?> watcher) {
    AbstractActorRunner<?> targetRunner = runners.get(target.actorId());
    AbstractActorRunner<?> watcherRunner = runners.get(watcher.actorId());
    if (targetRunner == null) throw new IllegalArgumentException("Unknown target actor: " + target.actorId());
    if (watcherRunner == null) throw new IllegalArgumentException("Unknown watcher actor: " + watcher.actorId());
    Consumer<Signal> listener = watcherRunner::signal;
    targetRunner.signalListeners.add(listener);
    return new WatchHandleImpl(targetRunner, listener);
}

/** Cancel a previously registered death watch. Idempotent. */
public void unwatch(WatchHandle handle) {
    handle.cancel();
}
```

**Add `watch`/`unwatch` to `BayouContext<M>` interface:**
```java
/**
 * Watch {@code target} for death. When it stops (crash or graceful), this actor receives
 * a {@link Terminated} signal via {@link com.cajunsystems.bayou.actor.Actor#onSignal}.
 *
 * @return a handle to cancel the watch
 */
WatchHandle watch(Ref<?> target);

/** Cancel a previously registered death watch. Idempotent. */
```

### Task 2 — Tests

Create `src/test/java/com/cajunsystems/bayou/DeathWatchTest.java`.

**Test cases:**

1. **`watch_receivesTerminated_onCrash`** — Actor A watches actor B via `system.watch(B, A)`. Actor B crashes (throws in handler). Assert A's `onSignal` is called with `Terminated("B")`.
   - Spawn A as an anonymous `Actor<String>` with `onSignal(signal, ctx)` that records signal via `AtomicReference<Signal>`.
   - Spawn B as `Actor<String>` that throws on any message.
   - Call `system.watch(refB, refA)`.
   - Send message to B to trigger crash.
   - Awaitility: assert A's recorded signal is `Terminated("B")`.

2. **`watch_receivesTerminated_onGracefulStop`** — Actor A watches actor B. Call `refB.stop()`. Assert A receives `Terminated("B")`.

3. **`unwatch_preventsTerminatedDelivery`** — Register watch, then `system.unwatch(handle)`. Stop B. Assert A's signal counter stays at 0.

4. **`watch_fromWithinActor_usingCtx`** — Actor A watches B using `ctx.watch(refB)` from within its `preStart` handler (or from first message handler). B is stopped. Assert A receives `Terminated("B")`.
   - This verifies the `BayouContext.watch()` path.

Use Awaitility for all timing assertions. Each test uses a fresh `BayouSystem`.
