# Phase 5, Plan 1: Death Spiral Guard

## Objective

Implement the restart-count guard in `OneForOneStrategy` and `AllForOneStrategy` so they return
`ESCALATE` once a child crashes more times than `RestartWindow.maxRestarts` within the rolling
time window. Wire up the ESCALATE case in `SupervisorRunner` so it actually propagates as a
crash signal to the supervisor's own parent (or logs a critical error and stops if top-level).
Clean up child registrations in `cleanup()` so a restarted supervisor can re-register its children
without a duplicate-ID collision.

## Context

### Relevant Accumulated Decisions

- `RestartDecision` has 4 values: RESTART, RESTART_ALL, STOP, ESCALATE
- `SupervisionStrategy` is `@FunctionalInterface` — supports lambda custom strategies
- Strategy `decide()` always returns constant for now — Phase 5 adds spiral counting
- `RestartWindow` stored in strategy constructor — signature stable for Phase 5
- `RestartWindow.UNLIMITED = new RestartWindow(Integer.MAX_VALUE, Duration.ZERO)`
- ESCALATE is a no-op stub in Phase 4 — Phase 5 implements actual tree propagation
- `AbstractActorRunner.stopFuture` is `volatile` — can be replaced in `restart()`
- `running.set(false)` in crash path — enables natural exclusion from RESTART_ALL sibling filter
- `SupervisorRunner.childRunners` is `CopyOnWriteArrayList` — thread-safe for `spawnChild`
- `BayouSystem.actors` is `ConcurrentHashMap` — concurrent remove from cleanup() is safe
- Crash signal fires after `cleanup()` and `stopFuture.complete()` — fully stopped first

### Key Design Points

**Spiral counting in strategies (per-child, per-strategy-instance):**

`decide()` in both strategies gains a `HashMap<String, ArrayDeque<Instant>> restartHistory`
instance field. On each call:
1. Short-circuit: if `restartWindow == RestartWindow.UNLIMITED`, return immediately
   (reference equality on the UNLIMITED constant is correct and sufficient).
2. Retrieve (or create) the deque for `childId`.
3. If `restartWindow.within()` is non-zero, trim entries older than `Instant.now().minus(within)`.
4. If `deque.size() >= restartWindow.maxRestarts()` → return `ESCALATE` (do NOT add to history).
5. Otherwise push `Instant.now()` onto the deque → return `RESTART` (or `RESTART_ALL`).

`AllForOneStrategy.decide()` uses the SAME logic, returning `RESTART_ALL` instead of `RESTART`.

**EscalationException (new, package-private):**

A plain `RuntimeException` subclass. Thrown from `escalate()` to crash the supervisor's `loop()`
naturally — the existing `catch (Exception e)` block in `loop()` picks it up, sets `terminalCause`,
calls `cleanup()`, completes `stopFuture`, then fires the crash signal to `crashListener` if
non-null (i.e., parent supervisor). No changes to `loop()` itself are needed.

**`AbstractActorRunner.escalate(Throwable cause)` (new, package-private):**

```
void escalate(Throwable cause) {
    if (crashListener == null) {
        context.logger().error(
            "Actor '{}': top-level supervisor exceeded restart window — stopping permanently",
            actorId, cause);
    }
    throw new EscalationException(actorId, cause);
}
```

Called only from `SupervisorRunner.processEnvelope()`. `crashListener` is the private field
already accessible within the class.

**`SupervisorRunner.processEnvelope()` ESCALATE case:**

Replace the current `logger.warn("ESCALATE not yet implemented...")` with:
```
case ESCALATE -> {
    logger.info("Supervisor '{}': child '{}' exceeded restart window — escalating",
        actorId, crash.actorId());
    escalate(crash.cause());
}
```

**`SupervisorRunner.cleanup()` — unregister children + clear list:**

After stopping all children, iterate `childRunners` and call
`context.system().unregisterActor(runner.actorId)` for each, then `childRunners.clear()`.
Needed so that a parent-driven `restart()` on this supervisor re-runs `initialize()`, which
calls `startAndRegister()` → `registerActor()` — which would fail with duplicate-ID exception
if children were never unregistered.

**`BayouSystem.unregisterActor(String actorId)` (new, package-private):**

One line: `actors.remove(actorId)`. `ConcurrentHashMap.remove` is safe from any thread.
`shutdown()` calls `actors.clear()` in its finally, so double-removal is harmless.

### Task 1 — EscalationException + strategy spiral counting

```java
package com.cajunsystems.bayou;

/** Thrown by a supervisor runner to propagate escalation up the supervision tree. */
final class EscalationException extends RuntimeException {
    EscalationException(String supervisorId, Throwable cause) {
        super("Supervisor '" + supervisorId + "' escalating: restart window exhausted", cause);
    }
}
```

**`OneForOneStrategy.java` — add spiral counting to `decide()`:**

1. Add import: `java.time.Instant`, `java.util.ArrayDeque`, `java.util.HashMap`, `java.util.Map`
2. Add field: `private final Map<String, ArrayDeque<Instant>> restartHistory = new HashMap<>()`
3. Replace `decide()` body:
```java
@Override
public RestartDecision decide(String childId, Throwable cause) {
    if (restartWindow == RestartWindow.UNLIMITED) return RestartDecision.RESTART;
    ArrayDeque<Instant> history =
        restartHistory.computeIfAbsent(childId, k -> new ArrayDeque<>());
    if (!restartWindow.within().isZero()) {
        Instant cutoff = Instant.now().minus(restartWindow.within());
        history.removeIf(t -> t.isBefore(cutoff));
    }
    if (history.size() >= restartWindow.maxRestarts()) {
        return RestartDecision.ESCALATE;
    }
    history.addLast(Instant.now());
    return RestartDecision.RESTART;
}
```

**`AllForOneStrategy.java` — identical logic, returns `RESTART_ALL`:**

Same imports, same field, same logic — final line returns `RestartDecision.RESTART_ALL`.

### Task 3 — DeathSpiralTest (3 tests)

**New file: `src/test/java/com/cajunsystems/bayou/DeathSpiralTest.java`**

```java
package com.cajunsystems.bayou;

import com.cajunsystems.bayou.actor.Actor;
import org.junit.jupiter.api.AfterEach;
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.Test;

import java.time.Duration;
import java.util.List;
import java.util.concurrent.TimeUnit;
import java.util.concurrent.atomic.AtomicInteger;

import static org.assertj.core.api.Assertions.assertThat;
import static org.awaitility.Awaitility.await;

class DeathSpiralTest {

    private BayouSystem system;

    @BeforeEach void setUp() throws Exception { system = BayouTestSupport.freshSystem(); }
    @AfterEach  void tearDown()               { system.shutdown(); }

    @Test
    void escalatesAfterExceedingRestartWindow() {
        // Window: 3 restarts in 10 seconds
        // Always-crashing child → 4th crash triggers ESCALATE → supervisor dies
        var preStarts = new AtomicInteger(0);
        Actor<String> alwaysCrashes = new Actor<>() {
            public void handle(String msg, BayouContext ctx) {}
            public void preStart(BayouContext ctx) {
                preStarts.incrementAndGet();
                throw new RuntimeException("always crashes");
            }
        };

        SupervisorRef sup = system.spawnSupervisor("sup", new SupervisorActor() {
            public List<ChildSpec> children() {
                return List.of(ChildSpec.stateless("crasher", alwaysCrashes));
            }
            public SupervisionStrategy strategy() {
                return new OneForOneStrategy(new RestartWindow(3, Duration.ofSeconds(10)));
            }
        });

        // Supervisor should die after 3 restarts (4 preStart calls total)
        await().atMost(10, TimeUnit.SECONDS).until(() -> !sup.isAlive());

        // Exactly 4 preStart calls: initial + 3 restarts; 4th crash triggers ESCALATE
        assertThat(preStarts.get()).isEqualTo(4);
        assertThat(system.lookup("crasher").map(ActorRef::isAlive)).contains(false);
    }

    @Test
    void unlimitedWindowNeverEscalates() throws Exception {
        // Crashes 3 times then succeeds — UNLIMITED window should never escalate
        var preStarts = new AtomicInteger(0);
        Actor<String> crashThreeTimes = new Actor<>() {
            public void handle(String msg, BayouContext ctx) {}
            public void preStart(BayouContext ctx) {
                if (preStarts.incrementAndGet() <= 3) {
                    throw new RuntimeException("crash #" + preStarts.get());
                }
            }
        };

        SupervisorRef sup = system.spawnSupervisor("sup", new SupervisorActor() {
            public List<ChildSpec> children() {
                return List.of(ChildSpec.stateless("worker", crashThreeTimes));
            }
            public SupervisionStrategy strategy() {
                return new OneForOneStrategy(RestartWindow.UNLIMITED);
            }
        });

        // Actor recovers on 4th preStart — both actor and supervisor stay alive
        await().atMost(5, TimeUnit.SECONDS).until(() ->
            system.lookup("worker").map(ActorRef::isAlive).orElse(false));

        assertThat(sup.isAlive()).isTrue();
        assertThat(preStarts.get()).isEqualTo(4);
    }

    @Test
    void topLevelEscalationStopsGracefully() {
        // Top-level supervisor with window=1; 2nd crash causes ESCALATE.
        // No parent → supervisor stops permanently; system remains usable.
        var preStarts = new AtomicInteger(0);
        Actor<String> alwaysCrashes = new Actor<>() {
            public void handle(String msg, BayouContext ctx) {}
            public void preStart(BayouContext ctx) {
                preStarts.incrementAndGet();
                throw new RuntimeException("always crashes");
            }
        };

        SupervisorRef sup = system.spawnSupervisor("sup", new SupervisorActor() {
            public List<ChildSpec> children() {
                return List.of(ChildSpec.stateless("crasher", alwaysCrashes));
            }
            public SupervisionStrategy strategy() {
                return new OneForOneStrategy(new RestartWindow(1, Duration.ofSeconds(10)));
            }
        });

        // Supervisor and child both stop; system does not throw
        await().atMost(5, TimeUnit.SECONDS).until(() -> !sup.isAlive());

        assertThat(sup.isAlive()).isFalse();
        assertThat(system.lookup("crasher").map(ActorRef::isAlive)).contains(false);

        // System remains operational — can still spawn new actors
        ActorRef<String> probe = system.spawn("probe", (msg, ctx) -> {});
        assertThat(probe.isAlive()).isTrue();
    }
}
```
