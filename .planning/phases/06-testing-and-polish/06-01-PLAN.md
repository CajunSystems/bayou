# Phase 6, Plan 1: Nested Supervisor Support

## Objective

Add `ChildSpec.supervisor()` so one supervisor can own another as a child, forming a proper
supervision tree. Wire up crash propagation so a child supervisor's escalation becomes a crash
signal in the parent supervisor's mailbox. Verify with two integration tests.

## Context

### Accumulated Decisions Relevant to This Plan

- `SupervisorRunner extends AbstractActorRunner<ChildCrash>` — mailbox typed to crash signals
- `cleanup()` unregisters children from `BayouSystem.actors` + clears `childRunners` — enables clean restart by parent
- `EscalationException extends RuntimeException` — thrown from `escalate()`, caught by existing `catch (Exception e)` in `loop()`, fires crash signal to parent
- `escalate()` on `AbstractActorRunner` — checks `crashListener == null` to detect top-level
- `ChildSpec` is a `sealed interface permits StatelessChildSpec, StatefulChildSpec, EventSourcedChildSpec` — adding `SupervisorChildSpec` requires updating the permits clause
- `SupervisorRunner.createChildRunner()` does an if-else chain on `ChildSpec` subtypes
- `startAndRegister()` calls `runner.toActorRef()` and registers it — must register `SupervisorRef` for nested supervisors

### Key Design

**`SupervisorChildSpec` (new, package-private):**

```java
record SupervisorChildSpec(String actorId, SupervisorActor supervisorActor) implements ChildSpec {}
```

**`ChildSpec.java` — two changes:**

1. Add `SupervisorChildSpec` to the `permits` clause:
   `public sealed interface ChildSpec permits StatelessChildSpec, StatefulChildSpec, EventSourcedChildSpec, SupervisorChildSpec`

2. Add factory method:
   ```java
   /** Creates a spec for a nested supervisor. */
   static ChildSpec supervisor(String actorId, SupervisorActor supervisorActor) {
       return new SupervisorChildSpec(actorId, supervisorActor);
   }
   ```

**`SupervisorRunner` — two changes:**

1. `createChildRunner()` — add branch for `SupervisorChildSpec`:
   ```java
   } else if (spec instanceof SupervisorChildSpec s) {
       return new SupervisorRunner(s.actorId(), context.system(),
               s.supervisorActor().strategy(),
               s.supervisorActor().children());
   }
   ```

2. `startAndRegister()` — register `SupervisorRef` when runner is a `SupervisorRunner`:
   ```java
   ActorRef<?> ref = (runner instanceof SupervisorRunner sr)
           ? sr.toSupervisorRef()
           : runner.toActorRef();
   context.system().registerActor(spec.actorId(), ref);
   return ref;
   ```
   (Replace the current `context.system().registerActor(spec.actorId(), runner.toActorRef())` call.)

This ensures `system.lookup("inner-sup")` returns the `SupervisorRef` and can be cast.

**Crash propagation mechanism (no new code needed):**

When a child `SupervisorRunner` escalates:
1. `escalate()` throws `EscalationException` inside the child supervisor's `processEnvelope()`
2. Child supervisor's `loop()` catches it → `terminalCause` set → `cleanup()` runs (stops + unregisters grandchildren) → `stopFuture.complete()` → crash signal fired to `crashListener`
3. `crashListener` is the parent supervisor's `this.tell(crash)` (set in `startAndRegister()`)
4. Parent supervisor receives `ChildCrash` for the child supervisor's actorId
5. Parent calls `strategy.decide(childSupervisorId, cause)` → RESTART / STOP / ESCALATE

Everything wires up through the existing crash machinery. No changes to `loop()` or crash signal flow needed.

### Task 2 — NestedSupervisionTest (2 tests)

**New file: `src/test/java/com/cajunsystems/bayou/NestedSupervisionTest.java`**

```java
package com.cajunsystems.bayou;

import com.cajunsystems.bayou.actor.Actor;
import org.junit.jupiter.api.AfterEach;
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.Test;

import java.time.Duration;
import java.util.List;
import java.util.concurrent.TimeUnit;
import java.util.concurrent.atomic.AtomicBoolean;

import static org.assertj.core.api.Assertions.assertThat;
import static org.awaitility.Awaitility.await;

class NestedSupervisionTest {

    private BayouSystem system;

    @BeforeEach void setUp() throws Exception { system = BayouTestSupport.freshSystem(); }
    @AfterEach  void tearDown()               { system.shutdown(); }

    @Test
    void nestedChildCrashHandledByChildSupervisor() throws Exception {
        // Grandchild crashes once then recovers. Parent should not be involved —
        // the child supervisor handles the restart internally.
        var crashed = new AtomicBoolean(false);

        Actor<String> crashOnce = new Actor<>() {
            public void handle(String msg, BayouContext ctx) {}
            public void preStart(BayouContext ctx) {
                if (crashed.compareAndSet(false, true)) {
                    throw new RuntimeException("grandchild first-start crash");
                }
            }
        };

        system.spawnSupervisor("parent", new SupervisorActor() {
            public List<ChildSpec> children() {
                return List.of(
                    ChildSpec.supervisor("child-sup", new SupervisorActor() {
                        public List<ChildSpec> children() {
                            return List.of(ChildSpec.stateless("grandchild", crashOnce));
                        }
                        public SupervisionStrategy strategy() {
                            return new OneForOneStrategy(RestartWindow.UNLIMITED);
                        }
                    })
                );
            }
            public SupervisionStrategy strategy() {
                return new OneForOneStrategy(RestartWindow.UNLIMITED);
            }
        });

        // Grandchild recovers after restart by its direct supervisor
        await().atMost(5, TimeUnit.SECONDS).until(() ->
            system.lookup("grandchild").map(ActorRef::isAlive).orElse(false));

        assertThat(system.lookup("child-sup").map(ActorRef::isAlive)).contains(true);
        assertThat(system.lookup("parent").map(ActorRef::isAlive)).contains(true);
    }

    @Test
    void childSupervisorEscalationPropagatesUpToParent() {
        // Child supervisor has a window of 1 restart; always-crashing grandchild
        // causes it to escalate after 2 crashes.
        // Parent strategy is STOP — child supervisor is permanently removed.
        Actor<String> alwaysCrashes = new Actor<>() {
            public void handle(String msg, BayouContext ctx) {}
            public void preStart(BayouContext ctx) {
                throw new RuntimeException("grandchild always crashes");
            }
        };

        SupervisorRef parent = system.spawnSupervisor("parent", new SupervisorActor() {
            public List<ChildSpec> children() {
                return List.of(
                    ChildSpec.supervisor("child-sup", new SupervisorActor() {
                        public List<ChildSpec> children() {
                            return List.of(ChildSpec.stateless("grandchild", alwaysCrashes));
                        }
                        public SupervisionStrategy strategy() {
                            // Window of 1: grandchild crashes 2 times → escalate
                            return new OneForOneStrategy(new RestartWindow(1, Duration.ofSeconds(10)));
                        }
                    })
                );
            }
            public SupervisionStrategy strategy() {
                // Parent permanently stops an escalating child supervisor
                return (childId, cause) -> RestartDecision.STOP;
            }
        });

        // Child supervisor escalates → parent stops it permanently
        await().atMost(10, TimeUnit.SECONDS).until(() ->
            system.lookup("child-sup").isEmpty());

        // Parent remains alive; child supervisor and grandchild are gone
        assertThat(parent.isAlive()).isTrue();
        assertThat(system.lookup("child-sup")).isEmpty();
        assertThat(system.lookup("grandchild")).isEmpty();
    }
}
```
