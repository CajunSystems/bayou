# Phase 15, Plan 1: TestKit

## Objective

`TestProbe<M>` is a spawnable actor that enqueues received messages and signals into
`LinkedBlockingQueue`s, exposing synchronous assertion methods: `expectMessage`,
`expectNoMessage`, `expectTerminated`. `TestKit.probe(system, id)` is the factory.
No Awaitility needed for straightforward actor tests — one line per assertion.

## Context

**Current state:** All actor primitives, supervision, timers, death watch, PubSub, and
persistent topics are implemented. Tests use Awaitility with CountDownLatches or AtomicIntegers.
No synchronous assertion API exists yet.

**Key APIs confirmed:**
- `system.watch(Ref<?> target, Ref<?> watcher)` → `WatchHandle` — test thread can call this
- `Actor.onSignal(Signal signal, BayouContext<M> ctx)` — probe's actor handles Terminated here
- `Terminated` record: `actorId()` → `String`
- `system.spawn(id, Actor<M>)` → `Ref<M>` — probe spawns its actor in constructor

**Key design decisions:**

- `TestProbe<M>` and `TestKit` live in `com.cajunsystems.bayou` (main source tree) — user-facing utilities for testing actors in downstream projects.
- `TestProbe<M>` holds:
  - `BayouSystem system` — for `watch()` calls
  - `LinkedBlockingQueue<M> messages` — regular messages
  - `LinkedBlockingQueue<Signal> signals` — Terminated / LinkedActorDied
  - `Ref<M> ref` — the spawned probe actor ref
- Internal `ProbeActor<M>` (package-private static nested or top-level class): implements `Actor<M>`, enqueues to `messages` in `handle()` and to `signals` in `onSignal()`.
- `expectMessage(M expected, Duration timeout)` — polls `messages`, checks equality via `Objects.equals`, throws `AssertionError` on timeout or mismatch. Returns the received value.
- `expectMessage(Class<T> type, Duration timeout)` — polls `messages`, checks `type.isInstance()`, throws `AssertionError` on failure. Returns the value cast to `T`.
- `expectNoMessage(Duration within)` — polls `messages` with timeout, throws `AssertionError` if anything arrives.
- `expectTerminated(Ref<?> ref, Duration timeout)` — calls `system.watch(ref, this.ref)` first (safe to call even if watch already set up — returns new WatchHandle, return value discarded), then polls `signals` queue, loops until `Terminated(ref.actorId())` arrives or timeout. Throws `AssertionError` on timeout.
- `InterruptedException` from `BlockingQueue.poll` is caught and rethrown as `RuntimeException` — test code stays clean.
- `TestKit` is a pure static factory (no instances):
  ```java
  public static <M> TestProbe<M> probe(BayouSystem system, String id) { ... }
  ```

**Create `src/main/java/com/cajunsystems/bayou/TestProbe.java`:**

```java
package com.cajunsystems.bayou;

import com.cajunsystems.bayou.actor.Actor;

import java.time.Duration;
import java.util.Objects;
import java.util.concurrent.LinkedBlockingQueue;
import java.util.concurrent.TimeUnit;

/**
 * A controllable actor probe for deterministic testing. Messages delivered to this probe
 * are available for synchronous assertion via {@link #expectMessage} and friends.
 *
 * <p>Obtain via {@link TestKit#probe(BayouSystem, String)}.
 *
 * @param <M> the message type this probe accepts
 */
public final class TestProbe<M> {

    private final BayouSystem system;
    private final Ref<M> ref;
    private final LinkedBlockingQueue<M> messages = new LinkedBlockingQueue<>();
    private final LinkedBlockingQueue<Signal> signals = new LinkedBlockingQueue<>();

    TestProbe(BayouSystem system, String id) {
        this.system = system;
        this.ref = system.spawn(id, new ProbeActor<>(messages, signals));
    }

    /** The {@link Ref} for this probe — use it to subscribe to topics, link, watch, or wire actors. */
    public Ref<M> ref() {
        return ref;
    }

    /**
     * Block until a message equal to {@code expected} arrives, or throw {@link AssertionError}
     * if the timeout elapses or a different message is received.
     *
     * @return the received message
     */
    public M expectMessage(M expected, Duration timeout) {
        M received = poll(messages, timeout);
        if (received == null) {
            throw new AssertionError(
                "expectMessage: timeout after " + timeout + " — expected: " + expected);
        }
        if (!Objects.equals(received, expected)) {
            throw new AssertionError(
                "expectMessage: expected [" + expected + "] but got [" + received + "]");
        }
        return received;
    }

    /**
     * Block until a message of the given {@code type} arrives, or throw {@link AssertionError}.
     *
     * @return the received message, cast to {@code T}
     */
    public <T extends M> T expectMessage(Class<T> type, Duration timeout) {
        M received = poll(messages, timeout);
        if (received == null) {
            throw new AssertionError(
                "expectMessage: timeout after " + timeout + " — expected type: " + type.getSimpleName());
        }
        if (!type.isInstance(received)) {
            throw new AssertionError(
                "expectMessage: expected type " + type.getSimpleName() +
                " but got " + received.getClass().getSimpleName() + ": " + received);
        }
        return type.cast(received);
    }

    /**
     * Assert that no message arrives within {@code within}. Throws {@link AssertionError}
     * if any message is received before the duration elapses.
     */
    public void expectNoMessage(Duration within) {
        M received = poll(messages, within);
        if (received != null) {
            throw new AssertionError(
                "expectNoMessage: unexpected message received: " + received);
        }
    }

    /**
     * Watch {@code ref} for termination, then block until a {@link Terminated} signal for
     * that actor arrives, or throw {@link AssertionError} on timeout.
     *
     * <p>Safe to call before stopping the watched actor — the watch is registered synchronously.
     */
    public void expectTerminated(Ref<?> watched, Duration timeout) {
        system.watch(watched, this.ref);
        long deadline = System.currentTimeMillis() + timeout.toMillis();
        while (System.currentTimeMillis() < deadline) {
            long remaining = deadline - System.currentTimeMillis();
            if (remaining <= 0) break;
            Signal signal;
            try {
                signal = signals.poll(remaining, TimeUnit.MILLISECONDS);
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
                throw new RuntimeException("Interrupted waiting for Terminated", e);
            }
            if (signal == null) break;
            if (signal instanceof Terminated t && t.actorId().equals(watched.actorId())) return;
        }
        throw new AssertionError(
            "expectTerminated: timeout after " + timeout + " — actor '" + watched.actorId() + "' did not terminate");
    }

    private M poll(LinkedBlockingQueue<M> queue, Duration timeout) {
        try {
            return queue.poll(timeout.toMillis(), TimeUnit.MILLISECONDS);
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new RuntimeException("Interrupted while waiting for message", e);
        }
    }
}
```

**Create `src/main/java/com/cajunsystems/bayou/ProbeActor.java`** (package-private):

```java
package com.cajunsystems.bayou;

import com.cajunsystems.bayou.actor.Actor;

import java.util.concurrent.LinkedBlockingQueue;

/** Internal actor backing a {@link TestProbe}. */
final class ProbeActor<M> implements Actor<M> {

    private final LinkedBlockingQueue<M> messages;
    private final LinkedBlockingQueue<Signal> signals;

    ProbeActor(LinkedBlockingQueue<M> messages, LinkedBlockingQueue<Signal> signals) {
        this.messages = messages;
        this.signals = signals;
    }

    @Override
    public void handle(M message, BayouContext<M> ctx) {
        messages.offer(message);
    }

    @Override
    public void onSignal(Signal signal, BayouContext<M> ctx) {
        signals.offer(signal);
    }
}
```

**Create `src/main/java/com/cajunsystems/bayou/TestKit.java`:**

```java
package com.cajunsystems.bayou;

/**
 * Factory for {@link TestProbe} — deterministic actor testing utilities.
 *
 * <pre>{@code
 * TestProbe<String> probe = TestKit.probe(system, "probe");
 * probe.ref().tell("hello");
 * probe.expectMessage("hello", Duration.ofSeconds(1));
 * }</pre>
 */
public final class TestKit {

    private TestKit() {}

    /**
     * Spawn a probe actor with the given {@code id} and return a {@link TestProbe} for it.
     * The probe's actor is registered in the system and stopped when the system shuts down.
     *
     * @param system the actor system
     * @param id     unique actor ID for this probe
     * @param <M>    the message type
     * @return a new {@code TestProbe}
     */
    public static <M> TestProbe<M> probe(BayouSystem system, String id) {
        return new TestProbe<>(system, id);
    }
}
```

### Task 2 — TestKitTest

**Create `src/test/java/com/cajunsystems/bayou/TestKitTest.java`** with 7 test cases.

Use same setup pattern as `StatelessActorTest` (SharedLogService + InMemoryPersistenceAdapter,
@BeforeEach/@AfterEach). **Do not use Awaitility** — use only `TestProbe` assertions.

**Test 1: `probeReceivesDirectTell`** — the done-when scenario, one line per assertion:
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
probe.ref().tell("hello");
probe.expectMessage("hello", Duration.ofSeconds(1));
```

**Test 2: `probeReceivesActorReply`** — actor sends to probe (no ctx.reply, just tell):
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
Ref<String> echo = system.spawn("echo", (msg, ctx) -> probe.ref().tell(msg + "-echo"));
echo.tell("ping");
probe.expectMessage("ping-echo", Duration.ofSeconds(1));
```

**Test 3: `expectMessageByType`** — type-based assertion:
```java
TestProbe<Object> probe = TestKit.probe(system, "probe");
probe.ref().tell("typed-message");
String msg = probe.expectMessage(String.class, Duration.ofSeconds(1));
assertThat(msg).isEqualTo("typed-message");
```

**Test 4: `expectNoMessagePassesWhenMailboxEmpty`** — assert silence:
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
probe.expectNoMessage(Duration.ofMillis(100));
```

**Test 5: `expectNoMessageFailsWhenMessageArrives`** — failure mode:
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
probe.ref().tell("unexpected");
// Give time for the message to reach the probe actor
Thread.sleep(50); // only for setup, not for assertion
assertThatThrownBy(() -> probe.expectNoMessage(Duration.ofSeconds(1)))
    .isInstanceOf(AssertionError.class)
    .hasMessageContaining("unexpected message");
```

**Test 6: `expectTerminatedDetectsActorStop`** — probe watches a dying actor:
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
Ref<String> dying = system.spawn("dying", (msg, ctx) -> {});
// expectTerminated sets up the watch, then we stop the actor
dying.stop(); // don't block — let it stop asynchronously
probe.expectTerminated(dying, Duration.ofSeconds(2));
```

**Test 7: `probeReceivesPubSubMessage`** — probe as PubSub subscriber:
```java
TestProbe<String> probe = TestKit.probe(system, "probe");
system.pubsub().subscribe("news", probe.ref());
system.pubsub().publish("news", "breaking");
probe.expectMessage("breaking", Duration.ofSeconds(1));
```
